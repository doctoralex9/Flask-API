# PYTHON_API

A simple Flask API with a basic HTML interface.

## Setup

1. Create a virtual environment:

    ```sh
    python -m venv venv
    ```

2. Activate the virtual environment:

    - On Windows:

        ```sh
        venv\Scripts\activate
        ```

    - On macOS/Linux:

        ```sh
        source venv/bin/activate
        ```

3. Install the dependencies:

    ```sh
    pip install -r requirements.txt
    ```

4. Run the application:

    ```sh
    python main.py
    ```

5. Open your browser and go to `http://127.0.0.1:5000` to see the interface.
